Functions

What are Functions?

At its core, a function is a reusable block of code that does a specific task. Functions are fundamental to programming, promoting modularity and code reuse. In Python, functions are first-class citizens, meaning they can be passed around and used as arguments like any other object (string, int, float, list, and so on).

Why Use Functions?

Functions offer a way to segment large and complex code bases into smaller, more manageable pieces. This makes the code easier to read, understand, and debug. It also avoids repetition and makes code reusable. Moreover, functions encapsulate logic, meaning if there's an error in a specific logic, you need to change it only in one place.

Defining and Calling Functions

In Python, functions are defined using the def keyword. After defining a function, it doesn't run on its own. You need to invoke or call the function to execute the code it contains.

def greet():
    print("Hello, World!")

To call this function, you simply use its name followed by parentheses: greet().

Parameters and Return Values

While some functions execute a task without needing any input, others might require one or more inputs to perform their task. These inputs are called parameters. Functions can also return values to be used elsewhere in the code. This is done using the return keyword.

def add(x, y):
    return x + y

The add function takes two parameters and returns their sum. If called with add(3, 4), it will return 7.

Types of Functions

There are primarily two types of functions in Python:

  • Built-in Functions: These are pre-defined in Python and are always available for use. Examples include len(), print(), and type().
  • User-defined Functions: These are defined by the users to meet specific needs. The above examples of greet() and add() are user-defined functions.

Variable-length Arguments

Sometimes, we may need to define functions that can accept any number of arguments. Python provides a way to handle this using *args for non-keyworded variable-length arguments and **kwargs for keyworded variable-length arguments.

Anonymous Functions

Python supports the creation of anonymous functions using the lambda keyword. These are small, throw-away functions without a name. They are useful when you need a small function for a short period and don't want to formally define it using def.

Exercises

  1. Create a function find_max that takes a list of numbers and returns the maximum number.
  2. Write a function calculate_length that takes a string as input and returns its length.
  3. Develop a function is_prime that checks if a number is prime or not.
  4. Design a function factorial that computes the factorial of a number.
  5. Implement a lambda function that multiplies two numbers together.

Solutions

  1. def find_max(numbers): return max(numbers)
  2. def calculate_length(s): return len(s)
  3. def is_prime(num): if num <= 1: return False for i in range(2, num): if num % i == 0: return False return True
  4. def factorial(n): if n == 0: return 1 return n * factorial(n-1)
  5. multiply = lambda x, y: x * y
