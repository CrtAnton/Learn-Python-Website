Decision Making in Python

If Statement

The if statement is used to test a condition. If the condition is true, the code inside the if block will run.

age = 17
if age < 18:
    print("You are a minor.")

This will output: You are a minor. if the age is less than 18.

If-Else Statement

The if-else statement tests a condition and executes the code inside the if block if the condition is true. Otherwise, it runs the code inside the else block.

age = 20
if age < 18:
    print("You are a minor.")
else:
    print("You are an adult.")

Given the age above, this will output: You are an adult.

If-Elif-Else Statement

The if-elif-else statement can be used to test multiple conditions. It goes through each condition until one is found to be true. If none are true, the else block is executed.

score = 85
if score >= 90:
    print("Grade A")
elif score >= 80:
    print("Grade B")
elif score >= 70:
    print("Grade C")
else:
    print("Grade D")

Given the score above, this will output: Grade B

Exercises

  1. Write an if statement that prints "Even" if a number is even and "Odd" otherwise.
  2. Given a variable called temperature, write an if-else statement that prints "It's hot!" if the temperature is above 30 and "It's cold!" otherwise.
  3. Write an if-elif-else statement that prints "Child", "Teen", or "Adult" based on the value of an age variable.

Solutions

  1. number = 4 # Example number if number % 2 == 0: print("Even") else: print("Odd")

  3. age = 15 # Example age if age <= 12: print("Child") elif age <= 18: print("Teen") else: print("Adult")
