Introduction to Python Programming

Print Statement

The print() function in Python is used to output information to the console. It's one of the most basic functions you'll use, especially for debugging.

print("Hello, World!")

This will output: Hello, World!

Mathematical Operations

Python supports all basic arithmetic operations.


a = 10
b = 20
print(a + b)  # Addition
print(a - b)  # Subtraction
print(a * b)  # Multiplication
print(a / b)  # Division

The code above performs basic arithmetic operations and outputs the results.

Variables

Variables are used to store data values. In Python, a variable is created the moment you first assign a value to it.


x = 5
y = "John"
print(x)
print(y)

The variable x will output 5 and y will output John.

Data Types

Python has various data types including integers, float (decimal), string (text), and boolean (True/False).


int_var = 5
float_var = 5.5
string_var = "Hello"
bool_var = True

The variables above represent different data types in Python.

Input

The input() function allows for user input. You can use it to get text input from the user.


name = input("Enter your name: ")
print("Hello, " + name + "!")

This code will prompt the user to enter their name and then greet them using the provided name.
