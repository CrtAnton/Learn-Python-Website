Lesson: Text Manipulation

Welcome to the fourth lesson of your Linux learning journey! In this lesson, you'll explore powerful text manipulation tools that enable you to work with and modify text data in various ways.

1. echo - Print Text to the Terminal

The echo command prints text to the terminal. It's often used for displaying messages or variables in scripts.

Example:

echo "Hello, Linux!"
echo "Today's date is $(date)"

In the first example, the text "Hello, Linux!" is printed to the terminal. In the second example, $(date) is a special construct that inserts the current date and time into the output.

2. printf - Format and Print Text

The printf command allows you to format and print text with greater control. It's particularly useful for aligning columns and formatting output.

Example:

printf "Name: %s\nAge: %d\n" "John" 25
printf "Decimal: %d\nFloating-point: %.2f\n" 10 3.14159

In the first example, %s and %d are placeholders for strings and integers, respectively. The values "John" and 25 are inserted in place of those placeholders. In the second example, %.2f specifies a floating-point number with two decimal places.

3. cut, paste - Manipulate Columns of Text

The cut command extracts specific columns of text from input, while the paste command combines lines from different files.

Example:

cut -f 1,3-5 data.txt
paste file1.txt file2.txt

In the cut example, -f 1,3-5 selects the first column and columns 3 to 5 from the "data.txt" file. In the paste example, the contents of "file1.txt" and "file2.txt" are combined side by side.

4. sed, awk - Advanced Text Manipulation using Patterns and Fields

sed and awk are powerful tools for advanced text manipulation. sed operates on patterns and offers find-and-replace capabilities, while awk is perfect for processing and extracting data from structured text.

Example:

sed 's/old/new/g' textfile.txt
awk '{print $2, $1}' data.txt

In the sed example, the command s/old/new/g searches for occurrences of "old" and replaces them with "new" globally. In the awk example, {print $2, $1} prints the second and first fields of each line in the "data.txt" file.

Exercises:

  1. Print the sentence "Linux is awesome!" using echo.
  2. Use printf to display your name and age.
  3. Extract the second and fourth columns from a file named "table.txt" using cut.
  4. Combine lines from "file1.txt" and "file2.txt" with a tab delimiter using paste.
  5. Replace all occurrences of "red" with "blue" in a file named "colors.txt" using sed.
  6. Use awk to print the third and first fields of each line in a file named "data.txt".

Solutions:

  1. echo "Linux is awesome!"
  2. printf "Name: %s\nAge: %d\n" "YourName" 30
  3. cut -f 2,4 table.txt
  4. paste -d "\t" file1.txt file2.txt
  5. sed 's/red/blue/g' colors.txt
  6. awk '{print $3, $1}' data.txt
