Lesson: Basic Command Line Navigation

Welcome to the second lesson of your journey into learning Linux and mastering the command line! In this lesson, you'll delve into fundamental command line navigation commands. These commands will allow you to navigate through your system, manage files and directories, and perform essential tasks efficiently.

1. ls - List Files and Directories

The ls command is your window into the contents of a directory. By default, when you type ls, it will list the files and directories in the current directory. However, you can also specify a path to list the contents of a different directory.

Example:

ls
ls /home/user/documents

2. cd - Change Directory

The cd command is your key to moving around the file system. It lets you change your current working directory to a specified path.

Example:

cd /home/user/documents
cd ..

3. pwd - Print Working Directory

The pwd command is like a GPS for your terminal. It tells you where you currently are in the file system by printing the full path of your current working directory.

Example:

pwd

4. mkdir - Create Directories

The mkdir command allows you to create new directories (folders). You simply provide the name of the directory you want to create.

Example:

mkdir new_folder

5. touch - Create Files

The touch command creates empty files. Just provide the name of the file you want to create.

Example:

touch new_file.txt

6. cp - Copy Files and Directories

The cp command lets you make duplicates of files and directories. You provide the source and destination paths.

Example:

cp file.txt /home/user/documents/
cp -r folder /backup/

7. mv - Move or Rename Files and Directories

The mv command can move files and directories to new locations or even rename them. You provide the source and destination paths.

Example:

mv old_file.txt new_location/
mv file.txt new_name.txt

8. rm - Remove Files and Directories

The rm command is used for deleting files and directories. Be cautious when using this command, as it doesn't send items to a "recycle bin"; they are gone for good.

Example:

rm file.txt
rm -r directory/

9. echo - Print Text to the Terminal

The echo command prints text to the terminal. It's often used for displaying messages or variables in scripts.

Example:

echo "Hello, Linux!"

10. man - Access Manual Pages

The man command provides access to manual pages for other commands. It's like an instruction manual, offering detailed information about how commands work and their available options.

Example:

man ls

11. clear - Clear the Terminal Screen

The clear command clears the terminal screen, giving you a fresh workspace.

Example:

clear

Exercises:

  1. List the contents of your home directory.
  2. Navigate to your Desktop directory.
  3. Create a new directory called "Scripts" on your Desktop.
  4. Create a file named "notes.txt" within the "Scripts" directory.
  5. Copy the "notes.txt" file to your home directory.
  6. Rename the "notes.txt" file to "important_notes.txt".

Solutions:

  1. ls ~
  2. cd ~/Desktop
  3. mkdir ~/Desktop/Scripts
  4. touch ~/Desktop/Scripts/notes.txt
  5. cp ~/Desktop/Scripts/notes.txt ~
  6. mv ~/notes.txt ~/important_notes.txt
