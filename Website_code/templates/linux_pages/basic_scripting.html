Lesson: Basic Scripting

Welcome to the fifth lesson of your Linux learning journey! In this lesson, you'll discover the basics of scripting, a powerful skill that enables you to automate tasks and create custom functionalities.

1. Variables and Data Types

Variables are like containers that hold different types of data. In scripts, you can use variables to store and manipulate information.

Example:

name="John"
age=25
greeting="Hello, $name!"
echo $greeting

In the above example, the variable name holds the value "John," and age holds the value 25. The variable greeting combines the string "Hello, " with the value of the name variable.

2. Arithmetic Operations

You can perform arithmetic operations using script variables. Linux supports addition, subtraction, multiplication, division, and more.

Example:

x=10
y=5
sum=$((x + y))
echo "Sum: $sum"

In this example, the variables x and y hold the values 10 and 5. The sum variable stores the result of the addition, and the $((...)) syntax calculates the sum of x and y.

3. Conditional Statements (if, elif, else)

Conditional statements help your script make decisions. Using if, elif, and else constructs, your script can execute different commands based on conditions.

Example:

age=18
if [ $age -ge 18 ]; then
    echo "You're an adult."
else
    echo "You're a minor."
fi

In this example, the script checks if the age variable is greater than or equal to 18. Depending on the result, it outputs whether the person is an adult or a minor.

4. Loops (for, while)

Loops help you repeat commands multiple times. The for loop iterates over a range of values, while the while loop continues executing as long as a condition is true.

Example:

for i in 1 2 3; do
    echo "Number: $i"
done

counter=0
while [ $counter -lt 5 ]; do
    echo "Count: $counter"
    counter=$((counter + 1))
done

The for loop iterates over the values 1, 2, and 3, and the while loop continues as long as the counter variable is less than 5. Inside the while loop, the counter is incremented by 1 in each iteration.

Scripting with vi Editor

The vi editor is a versatile tool for creating and editing scripts. To start, type vi script.sh in the terminal, where "script.sh" is the name of your script file.

Inside vi:

i (for Insert Mode)
# Your script content here
[ESC] (to exit Insert Mode)
:wq (to save and exit)

Use the "i" key to enter Insert Mode, where you can type your script. Press [ESC] to exit Insert Mode. To save and exit, type ":wq" and press [Enter].

Exercises:

  1. Create a script that calculates and prints the area of a rectangle.
  2. Write a script that checks if a given number is even or odd.
  3. Use a loop to print numbers from 1 to 10.

Solutions:

  1. #!/bin/bash
length=5
width=3
area=$((length * width))
echo "Area of rectangle: $area"
  2. #!/bin/bash
number=7
if [ $((number % 2)) -eq 0 ]; then
    echo "Even"
else
    echo "Odd"
fi
  3. #!/bin/bash
for i in {1..10}; do
    echo $i
done
