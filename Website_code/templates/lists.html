Lists in Python

Introduction to Lists

A list in Python is an ordered collection of items which can contain any type of object, including other lists. Lists are mutable, meaning their contents can be changed after they are created.

Creating Lists

There are several ways to create lists in Python:

  • Using square brackets: fruits = ["apple", "banana", "cherry"]
  • Using the list constructor: numbers = list((1, 2, 3))
  • Using list comprehension: squares = [x**2 for x in range(5)]

The most common method is using square brackets, but other methods can be useful in specific scenarios.

List Indexing

Items in a list can be accessed using their index, which starts from 0 for the first element.


colors = ["red", "blue", "green"]
print(colors[0])  # Outputs: red
print(colors[2])  # Outputs: green

In the above example, we access the first and third elements of the list using their indices. Lists also support negative indexing, where -1 refers to the last item, -2 refers to the second last item, and so on.

List Manipulation

Python lists come with various methods that allow you to modify and manipulate them.

append()

The append() method adds an item to the end of the list.


animals = ["cat", "dog"]
animals.append("elephant")
print(animals)  # Outputs: ['cat', 'dog', 'elephant']

remove()

The remove() method removes the first occurrence of a value from the list.


numbers = [1, 2, 3, 2, 4]
numbers.remove(2)
print(numbers)  # Outputs: [1, 3, 2, 4]

pop()

The pop() method removes and returns the item at the specified index. If no index is specified, it removes and returns the last item.


fruits = ["apple", "banana", "cherry"]
popped_fruit = fruits.pop(1)
print(popped_fruit)  # Outputs: banana
print(fruits)        # Outputs: ['apple', 'cherry']
