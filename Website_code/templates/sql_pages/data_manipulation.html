Data Manipulation Statements

SQL offers three primary statements for data manipulation:

  • INSERT Statement: Adds new rows to a table.
  • UPDATE Statement: Modifies existing data in a table.
  • DELETE Statement: Removes data from a table.

Example: INSERT Statement

To insert a new book into the "Books" table:

INSERT INTO Books (Title, Author, PublicationYear)
VALUES ('The Great Gatsby', 'F. Scott Fitzgerald', 1925);

Example: UPDATE Statement

To update the publication year of a book:

UPDATE Books
SET PublicationYear = 2020
WHERE Title = 'The Great Gatsby';

Example: DELETE Statement

To delete a book from the table:

DELETE FROM Books
WHERE Title = 'The Great Gatsby';

Data Integrity and Constraints

Data integrity ensures accuracy and consistency in databases. Constraints are rules applied to tables to enforce data integrity.

  • Primary Key Constraint: Ensures each row is uniquely identifiable.
  • Foreign Key Constraint: Establishes relationships between tables.
  • Unique Constraint: Enforces uniqueness for values in a column.
  • Check Constraint: Defines conditions for data entry.
