Learning SQL Fundamentals

Understanding SQL Queries

SQL queries are used to retrieve specific data from a database. They allow you to interact with the database by instructing it to retrieve, filter, and order data based on your requirements. Let's break down the key components of a SQL query:

  • SELECT Clause: Specifies which columns you want to retrieve from a table.
  • FROM Clause: Specifies the table from which you're retrieving data.
  • WHERE Clause: Filters data based on specific conditions.
  • ORDER BY Clause: Sorts the results in ascending or descending order.

Writing Simple Queries

Consider a hypothetical database with a table named "Employees." This table contains information about employees in a company, including columns like "EmployeeID," "FirstName," "LastName," "Department," and "Salary."

Example 1: Retrieve All Employee Names

To retrieve the first and last names of all employees:

SELECT FirstName, LastName
FROM Employees;

Example 2: Filter Employees by Department

To retrieve employees from the "Sales" department:

SELECT FirstName, LastName
FROM Employees
WHERE Department = 'Sales';

Example 3: Sorting Employees by Salary

To retrieve employees sorted by salary in descending order:

SELECT FirstName, LastName, Salary
FROM Employees
ORDER BY Salary DESC;

Exercises: Practice Your Skills

  1. Retrieve the names of all employees who have a salary greater than $50,000.
  2. Retrieve the first name, last name, and department of employees from the "Marketing" department.
  3. Retrieve the names of employees and their salaries, ordered by salary in ascending order.
  4. Retrieve the first name, last name, and department of employees with IDs 101 and 105.
  5. Retrieve the first name and last name of employees whose last name starts with the letter "S".

Exercise Solutions:

Solutions to the exercises:

 FROM Employees
 ORDER BY Salary ASC;
