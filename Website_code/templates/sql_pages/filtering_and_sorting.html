Mastering Filtering, Sorting, and Combining Conditions in SQL

Filtering Data with Comparison Operators and Wildcards

Comparison operators and wildcards are crucial for filtering data in SQL.

  • Comparison Operators: Use operators like =, >, <, etc. to compare values.
  • Wildcards: Use % and _ with the LIKE operator to match patterns within strings.

Example: Filtering Using Comparison Operators

SELECT FirstName, LastName
FROM Employees
WHERE Salary > 50000;

Example: Filtering Using Wildcards

SELECT FirstName, LastName
FROM Customers
WHERE Email LIKE '%@gmail.com';

Combining Conditions with Logical Operators

Logical operators help you create complex conditions by combining filters.

  • AND: Combines conditions that must all be true.
  • OR: Combines conditions where at least one must be true.

Example: Combining Conditions with AND and OR

SELECT FirstName, LastName
FROM Employees
WHERE Department = 'Sales'
AND Salary > 60000;

Sorting Query Results with ORDER BY

The ORDER BY clause is used to sort query results.

SELECT FirstName, LastName, Salary
FROM Employees
ORDER BY Salary DESC;

Exercises: Practice Your Skills

  1. Retrieve books with titles starting with the letter "A."
  2. Retrieve customers whose first name is "John" or last name is "Doe."
  3. Retrieve employees from the "HR" department with a salary less than $50,000.
  4. Retrieve customers who have a Gmail address and are from the "Marketing" department.
  5. Retrieve employees sorted by department in ascending order and then by salary in descending order.

Exercise Solutions:

Solutions to the exercises:

1. SELECT Title
 FROM Books
 WHERE Title LIKE 'A%';

2. SELECT FirstName, LastName
 FROM Customers
 WHERE FirstName = 'John' OR LastName = 'Doe';

3. SELECT FirstName, LastName
 FROM Employees
 WHERE Department = 'HR' AND Salary < 50000;

4. SELECT FirstName, LastName
 FROM Customers
 WHERE Email LIKE '%@gmail.com' AND Department = 'Marketing';

5. SELECT FirstName, LastName, Department, Salary
 FROM Employees
 ORDER BY Department ASC, Salary DESC;
