Mastering Filtering, Sorting, and Combining Conditions in SQL

Filtering and sorting data are essential skills when working with databases. In this lesson, we'll delve into the world of SQL operators and wildcards, teaching you how to filter data effectively, combine conditions using logical operators, and sort query results for better readability.

Filtering Data with Comparison Operators and Wildcards

Comparison operators allow you to compare values and make decisions based on the results. SQL supports common comparison operators such as =, >, <, >=, and <=. Additionally, wildcards like % and _ are used in combination with the LIKE operator to match patterns within strings.

Example: Filtering Using Comparison Operators

Suppose you have an "Employees" table with a "Salary" column. To retrieve employees with a salary greater than $50,000:

SELECT FirstName, LastName
FROM Employees
WHERE Salary > 50000;

Example: Filtering Using Wildcards

Imagine you have a "Customers" table with an "Email" column. To retrieve customers with Gmail addresses:

SELECT FirstName, LastName
FROM Customers
WHERE Email LIKE '%@gmail.com';

Combining Conditions with Logical Operators

Logical operators (AND, OR, NOT) help you create complex conditions by combining multiple filters.

Example: Combining Conditions with AND and OR

SELECT FirstName, LastName
FROM Employees
WHERE Department = 'Sales'
AND Salary > 60000;

Sorting Query Results with ORDER BY

The ORDER BY clause is used to sort query results in ascending (ASC) or descending (DESC) order.

SELECT FirstName, LastName, Salary
FROM Employees
ORDER BY Salary DESC;

Exercises: Practice Your Skills

  1. Retrieve books with titles starting with the letter "A."
  2. Retrieve customers whose first name is "John" or last name is "Doe."
  3. Retrieve employees from the "HR" department with a salary less than $50,000.
  4. Retrieve customers who have a Gmail address and are from the "Marketing" department.
  5. Retrieve employees sorted by department in ascending order and then by salary in descending order.
