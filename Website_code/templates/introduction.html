Introduction to Python

Printing in Python

In Python, the print() function is used to display messages, data, or the output of your code to the console. The function allows you to display strings, numbers, and other data types:

print("Hello, World!")
print(5)
print(3.14)

Math Operations

Python provides a set of operators for basic arithmetic calculations:


a + b   # Addition
a - b   # Subtraction
a * b   # Multiplication
a / b   # Division
a % b   # Modulus (Remainder)
a ** b  # Exponentiation
a // b  # Floor division

For example, if \(a = 5\) and \(b = 2\), then \(a / b\) will give 2.5, but \(a // b\) will give 2.

Variables

Variables are named memory locations used to store data values. Once a variable is stored, the space in the memory is reserved for that variable:


name = "John"
age = 30
pi = 3.14

Here, "John" is a string, 30 is an integer, and 3.14 is a floating-point number.

Data Types

Python supports several built-in data types. Some of the most common ones are:

  • int: Represents integer values, e.g., 1, 100, -30.
  • float: Represents floating point numbers, e.g., 3.14, -0.001.
  • str: Represents string values, e.g., "Hello", 'Python'.
  • list: An ordered collection of items, e.g., [1, 2, 3, "apple"]
  • tuple: Similar to a list but immutable (cannot be changed), e.g., (1, 2, 3)
  • dict: An unordered collection of key-value pairs, e.g., {"name": "John", "age": 30}

User Input

The input() function allows you to get input from the user. By default, this function returns a string:

name = input("Enter your name: ")
age = int(input("Enter your age: "))

In the above example, we use the int() function to convert the input string to an integer.
