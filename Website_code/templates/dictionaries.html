Dictionaries in Python

Introduction to Dictionaries

A dictionary in Python is an unordered collection of data stored as a pair of a key and its corresponding value. Dictionaries are mutable, meaning their elements can be changed after they are created.

Creating Dictionaries

Dictionaries can be created using curly braces `{}` or the built-in `dict()` function.


# Using curly braces
person = {"name": "Alice", "age": 30}

# Using the dict() function
car = dict(brand="Ford", model="Mustang", year=1964)

In the first example, we define a dictionary `person` with keys "name" and "age" and their respective values. In the second example, we use the `dict()` function to create a dictionary for a car.

Dictionary Manipulation

Python dictionaries come with a variety of methods that allow you to modify and manipulate them.

Accessing Values

You can access the value for a particular key using square brackets or the `get()` method.


print(person["name"])      # Outputs: Alice
print(car.get("model"))    # Outputs: Mustang

Adding or Updating Key-Value Pairs

Add a new key-value pair or update the value for an existing key using square brackets.


person["job"] = "Engineer"
car["year"] = 2020

Removing Key-Value Pairs

Use the `pop()` method to remove a key-value pair by its key.


person.pop("age")
